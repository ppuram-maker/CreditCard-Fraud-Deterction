# Credit Card Fraud Detection

## Project Overview
This project uses Machine Learning (XGBoost) to detect fraudulent credit card transactions.

## Features
- Data Preprocessing
- Handling Imbalanced Data (SMOTE)
- Feature Scaling
- XGBoost Model
- ROC Curve & Evaluation Metrics

## Dataset
Dataset contains anonymized transaction features (V1–V28), Time, Amount, and Class.

## How to Run
1. Install libraries:
   pip install -r requirements.txt

2. Open notebook:
   creditcard_fraud_detection_system.ipynb

## Results
- Accuracy: ~99.9%
- High Recall for Fraud Detection
